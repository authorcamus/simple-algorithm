给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.

----

## 三指针

一个走到链表尾部tail，一个走到倒数第N个 cur，一个prev指向

```javascript
var removeNthFromEnd = function(head, n) {
    if (head == null){
        return head
    }
    let p1 = head
    let p2 = head

    while (n > 0){
        p1 = p1.next; n--;
    }
    while (p1 && p1.next){
        p1 = p1.next
        p2 = p2.next
    }
    if (p1 == null){
        p2 = p2.next
        return p2
    } else {
        p2.next = p2.next.next
    }
    return head
};
```
