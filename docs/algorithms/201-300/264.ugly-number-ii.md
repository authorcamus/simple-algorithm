
编写一个程序，找出第 n 个丑数。

丑数就是只包含质因数 2, 3, 5 的正整数。


```cpp
示例:

输入: n = 10
输出: 12
解释: 1, 2, 3, 4, 5, 6, 8, 9, 10, 12 是前 10 个丑数。
说明:  

1 是丑数。
n 不超过1690。

```

---

三个指针，表示乘上2，3，5得到最小数的起始位置

```cpp
class Solution {
public:
    int nthUglyNumber(int n) {
        int u[n];
        u[0] = 1;
        int p1 = 0, p2 = 0, p3 = 0;// 表示 最后一个质因子是2，3，5的最小值下标
        for (int i = 1; i < n; i++){
            u[i] = min(u[p1] * 2, min(u[p2] * 3,u[p3] * 5));
            if (u[i] == u[p1] * 2) p1++;
            if (u[i] == u[p2] * 3) p2++;
            if (u[i] == u[p3] * 5) p3++;
        }
        return u[n-1];
    }
};
```
