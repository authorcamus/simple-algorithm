给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

你可以假设数组中无重复元素。

---

## 二分法

典型的二分模板代码

```javascript
var searchInsert = function(nums, target) {
    let l = 0, r = nums.length // 插入位置 从0 到 n,包括n
    while (l < r){
        let mid = (l+r) >> 1
        if (nums[mid] >= target) r = mid
        else l = mid+1
    }
    return l
};
```
