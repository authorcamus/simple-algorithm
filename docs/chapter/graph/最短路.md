# 最短路

最短路算法的分类：

- 单源最短路
  - 所有边权都是正数
    - 朴素的Dijkstra算法 O(n^2) 适合稠密图
    - 堆优化版的Dijkstra算法 O(mlog n) （m是图中节点的个数）适合稀疏图
  - 存在负权边
    - Bellman-Ford O(nm)
    - spfa 一般O(m),最坏O(nm)
- 多源汇最短路  Floyd算法 O(n^3)
