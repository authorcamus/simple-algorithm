# 排序算法

我们常说八大排序算法，实际上排序有非常多的算法，一般我们只需要掌握三种即可：冒泡、快排、归并

其中冒泡最简单，我们直接给出代码：

冒泡排序

```javascript
function bubbleSort(nums) {
    for (let i = 0; i < nums.length - 1; i++) { // len - 1次冒泡
        for (let j = 0; j < nums.length - i - 1; j++) { // 依次比较相邻元素，进行冒泡，比较区间[0,len - 1 - i]
            if (nums[j] > nums[j + 1]) {
                let tmp = nums[j]
                nums[j] = nums[j+1]
                nums[j+1] = tmp
            }
        }
    }
    return nums
}
```
